# Customer Churn Analysis using Machine Learning

## 📌 Project Overview
Customer churn is one of the biggest challenges faced by businesses today.  
This project focuses on analyzing customer behavior and predicting whether a customer is likely to leave the company using Machine Learning techniques.

The project includes:
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization
- Machine Learning Model Building
- Model Evaluation
- Business Insights & Recommendations

---

## 🚀 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- XGBoost
- Jupyter Notebook

---

## 📂 Dataset Features

The dataset contains customer-related information such as:

- Customer ID
- Credit Score
- Country
- Gender
- Age
- Balance
- Number of Products
- Active Membership Status
- Estimated Salary
- Churn Status

---

## 📊 Exploratory Data Analysis (EDA)

Performed detailed EDA to understand customer behavior patterns:
- Churn Distribution Analysis
- Gender-wise Churn Analysis
- Country-wise Customer Distribution
- Correlation Analysis
- Feature Importance Visualization

---

## ⚙️ Machine Learning Models Used

### 1. Logistic Regression
- Baseline classification model
- Simple and interpretable

### 2. Random Forest Classifier
- Ensemble learning technique
- Handles feature importance effectively

### 3. XGBoost Classifier
- Advanced boosting algorithm
- High-performance predictive model

---

## 💡 Key Business Insights

- Older customers showed higher churn probability.
- Active members were less likely to churn.
- Customers with fewer products had increased churn rates.
- Balance and credit score significantly influenced churn behavior.
